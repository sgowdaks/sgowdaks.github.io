My favorite feeling is when something I built finally runs just right — and I immediately start thinking about how to make it better. GitHub Repo PyPI (pip install nichirin)

It integrates Apache Spark for parallel processing to scale web crawling and indexing, and includes preprocessing steps to structure raw inputs for downstream embedding and retrieval.

Platform Features:

  • Multi-level crawling with configurable depth and rate limits.
  • Apache Solr-based indexing for fast full-text retrieval.
  • Spark integration for parallel processing and scalable indexing.
  • Preprocessing strategies to clean and structure raw HTML/text for embeddings.
  • Published to PyPI (pip install nichirin) and open sourced on GitHub.
